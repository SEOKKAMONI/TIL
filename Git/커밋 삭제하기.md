<h1>커밋 삭제하는 방법</h1>

1. 커밋 삭제
    - 커밋을 삭제하는거는 간단함 git reset을 써주면 된다.<br/>
    최근 커밋 한개 를 취소하려면 `git reset HEAD^`를 써주면 된다.<br/>
    또한 여러개를 삭제하려면 `git reset HEAD~개수`를 입력하면 된다.<br/>
    예시를 보여주겠음
