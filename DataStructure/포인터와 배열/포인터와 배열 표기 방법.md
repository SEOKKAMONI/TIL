<h1>포인터와 배열 표기 방법</h1>

1. 포인터와 배열 간단한 예시 풀이
    ```
        int a[5];
        int* b;
        b = a; // 배열 a의 시작 주소인 a[0]의 주소를 포인터 변수 b의 저장
    ```
    - 따라서 배열의 이름은 배열의 첫번째 요소의 주소이다
